This is the first public release of fusion-lib.

## Scope

- JavaScript utilities for UI behavior, dialogs, backend calls, tooltips, and calendars
- PHP helper classes for dates, money, validation, and backend glue
- Custom ClassLoader without Composer dependency
- Explicit documentation of design philosophy

## Guarantees

- No framework dependencies
- No build tooling required
- Stable public interfaces
- Manual, explicit integration

## Non-Goals

- No SPA support
- No npm / Composer packages
- No automated dependency resolution

fusion-lib v1.0.0 represents a stable baseline intended for long-term use.
